什么是 HTML？

·HTML 指的是超文本标记语言(Hyper Text Markup Language),HTML 不是一种编程语言，而是一种标记语言(markup language),标记语言是一套标记标签markup tag),HTML 使用标记标签来描述网页

HTML 标签

·HTML 标记标签通常被称为 HTML 标签 (HTML tag),标签对中的第一个标签是开始标签，第二个标签是结束标签,HTML 标签标准写法成对出现

HTML 元素

·HTML 元素指的是从开始标签（start tag）到结束标签（end tag）的所有代码。

·<p> 这是一个段落 </p>

·<p> 为开始元素； "这是一个段落" 为元素内容 </p>为结束元素

·在开始标签中添加斜杠，比如 <br />，是关闭空元素的正确方法，HTML、XHTML 和 XML 都接受这种方式。

例如：

·<html> 与 </html> 之间的文本描述网页

·<body> 与 </body> 之间的文本是可见的页面内容

·<h1> 与 </h1> 之间的文本被显示为标题

·<p> 与 </p> 之间的文本被显示为段落

HTML 属性

·属性总是在 HTML 元素的开始标签中规定。HTML 标签可以拥有属性。属性提供了有关 HTML 元素的更多的信息。

例如：

·<a href="http://www.w3school.com.cn">This is a link</a>

·http为属性名称 http://www.w3school.com.cn为属性值

·大多数HTML元素的属性有： class，id，style，title，如需更多关于标准属性的信息，请访问： HTML 标准属性参考手册
